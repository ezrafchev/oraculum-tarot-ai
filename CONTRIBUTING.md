# Contribuindo

Obrigado por ajudar a tornar o ORACULUM AI mais confiável, acessível e transparente.

## Princípios

- Preserve o sorteio criptográfico independente da interpretação.
- Não introduza chaves, tokens ou serviços pagos como requisito.
- Trate o Tarô como linguagem simbólica, sem afirmar fatos privados ou diagnósticos.
- Mantenha o núcleo utilizável offline e sem cadastro.
- Inclua testes para alterações no baralho, motor, persistência ou auditoria.

## Fluxo de contribuição

1. Crie uma branch curta a partir de `main`.
2. Faça mudanças focadas e documentadas.
3. Execute `npm run lint`, `npm run typecheck`, `npm run test` e `npm run build:pages`.
4. Abra um pull request explicando impacto, decisão técnica e validações.

Discussões devem seguir o [Código de Conduta](CODE_OF_CONDUCT.md). Vulnerabilidades devem seguir [SECURITY.md](SECURITY.md), sem exposição pública antecipada.
